# ZK & ML 编译器工程师

## 关于我们
我们是零知识机器学习技术的先驱，开发突破性的解决方案，将隐私保护计算与先进的机器学习能力相结合。我们的编译器团队在使zkML实用化和高效化方面发挥着关键作用。

## 职位概述
我们正在寻找一位在零知识证明和密码学方面具有专业知识的优秀编译器工程师，负责开发专门用于ZK机器学习应用的编译器。该职位将密码学协议、机器学习和编译器优化三个领域紧密结合。
